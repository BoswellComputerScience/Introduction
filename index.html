Welcome to Boswell Computer Science UIL

This website will host documentation and links to tutorials to help you study for UIL Computer Science. If you're looking to contribute to the site, see Contributing to the Site.

Getting Started

Creating an Account

Go to https://github.com and fill in the username, email, and password fields. Click Sign up for Github

Link your account

Fill in the form to connect your account to the Boswell Computer Science GitHub.

View Lessons and Source Code

View lessons written in markup and source code examples for UIL problems

Contribute

Think you have a good lesson? Or maybe you've written some particularly good code. Contribute to the Boswell Computer Science organization to receive feedback and tips.
